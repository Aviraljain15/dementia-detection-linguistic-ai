# Dementia Detection using Speech Analysis

This project implements a hybrid AI system for detecting dementia using linguistic biomarkers and machine learning models.

## Features
- Linguistic feature extraction (PIU, pause rate, etc.)
- TF-IDF baseline
- XGBoost model
- SHAP explainability

## Dataset
DementiaBank Pitt Corpus (processed)

## Results
- Accuracy: 87%
- AUC: 0.9451

## How to Run
pip install -r requirements.txt
python main.py
